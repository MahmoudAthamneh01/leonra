## 🚀 كيفية التشغيل

### التطوير السريع
```bash
# تثبيت التبعيات
npm install

# تشغيل Frontend + Backend معاً
npm run full:dev

# أو منفصل
npm run dev              # Frontend على 3000
npm run server:dev       # Backend على 3001
```

### الإنتاج
```bash
# إعداد المشروع
npm run setup

# بناء المشروع
npm run full:build

# النشر
npm run deploy
```
